# 更新历史

## 2025.12.11 v3.9.1

- 修复自动更新问题

## 2025.12.11 v3.9.0

- 增加自动更新浏览器的功能。

## 2016.01.28 v3.7.3

- 修正 Stable/Beta/Dev/Canary 分支的更新问题。

## 2015.11.05 v3.7

- 修正 64 位 MyChrome 找不到本地安装版 Chrome 的问题。
- 增加鼠标双击关闭标签页，右键关闭标签页功能（按住 Shift 键可恢复右键菜单）。
- 增加保留最后一个标签页、滚轮切换标签页功能。

## 2015.10.11 v3.6

- 解决近期 Canary 无法更新问题。
- 代理设为“跟随系统”时支持 pac。
- 增加老板键功能：默认 Alt+z 或鼠标右键双击隐藏浏览器并静音（需 Win Vista+）。

## 2015.08.17 v3.5.1

- 修正简体中文情况下从离线安装包提取浏览器无效的问题。

## 2015.08.14 v3.5

- 增加多语言支持，内置简体中文、繁体中文、英语。
- 其它小的优化。

## 2015.08.03 v3.4

- 修正 Chromium 在 Win 10 中锁定任务栏后出现双图标的问题。
- 代理设置增加“直接连接”、“跟随系统”选项。
- 优化 IP 验证的速度。

## 2015.07.22 v3.3

- 修正 Win 10 中锁定任务栏后出现双图标的问题。从 MyChrome 3.3 开始，Win 7 以上系统中可以直接右键菜单锁定 MyChrome.exe 至任务栏。
- 增加从新浪更新 Chrome 浏览器的选项（仅支持 Stable/Beta/Dev 分支）。

## 2015.07.02 v3.2.1

- 去掉网络扩展 inet.exe，减小软件体积（压缩包 700K 左右）。
- inet.exe 以插件方式提供：如需 SOCKS 代理支持或查找 IP 用于浏览器访问部分 google 服务，可在设置界面中勾选“启用网络增强插件”自动下载插件。若不想用插件，取消勾选或直接删除 inet.exe、MSVCR100.dll 这两个文件。

## 2015.06.05 v3.1.3

- 增加对 SOCKS4、SOCK5 代理的支持。
- 增加自主查找 google 系 IP 的功能，MyChrome 中设代理服务器为 google.com，且勾选“恢复浏览器部分 google 服务”，不使用代理软件也可在浏览器中访问部分 google 服务。第一次使用或很久未使用过 MyChrome 情况下，可能暂时不能访问 google，可过几分钟待找到相应 IP 后再重启浏览器试试。如访问www.google.com.hk有困难，可试试www.google.com/ncr。
- 修正最近 Chrome 新版重装系统或复制到其它电脑丢失扩展的问题。

## 2015.02.11 v3.0.2

- 没有代理软件也可更新浏览器：在设置界面中代理服务器选择“google.com”即可。修正上一版本中，不能下载保存在国内非 google 服务器的更新文件的问题。
- 设置界面点击最新版本号可显示浏览器更新链接，可根据网络情况自主选择不同链接下载更新。

## 2015.02.04 v3.0.1

- 修正 64 位系统中浏览器退出后关闭外部程序时，残留子进程的问题（比如 GoAgent 的 python27.exe）。
- 在 MyChrome 设置界面点击“最新版本号”，可显示所有下载链接。
- 没有代理软件也可更新浏览器。现在将代理服务器设为“google.com”，可自动获取 google 可用 IP 作为代理（IP 来自网络）。

## 2015.01.29 v2.9.6

- 修正近期通过 GoAgent 更新获取的文件大小为 0、更新失败的问题。
- 修正本地 Chromium revision 显示错误影响更新的问题。
- 暂时解决在不同电脑运行扩展全部丢失的问题。
- MyChrome 自身更新平台移至 TaoCode（阿里开源平台）。

## 2014.08.30 v2.9.5

- 增加 64 位 Chrome 稳定版（Stable）更新支持。

## 2014.06.18 v2.9.4

- 修正 2.9.3 版一处逻辑错误，这一错误导致勾选 32-bit(x86)情况下代{过}{滤}理设置不起作用。
- 获取更新信息过程全部改用 SSL 加密，避免传输受干扰。
- 增加 omaha protocol v2 方式，作为 32 位浏览器获取更新的备用方式。

## 2014.06.07 v2.9.3

- 按操作系统自动选择 32-bit 或 64-bit 浏览器，64-bit Win 7 以上系统优先下载 64-bit Chrome。
- 如果在设置中选择"32-bit (x86)"，则忽略操作系统区别，只下载 32-bit 浏览器。
- MyChrome 设置快捷方式改为"MyChrome.vbs"（去掉设置二字）。

## 2014.05.01 v2.9.2

- 修正对 Canary 64 位版本的支持。
- 修正外部程序参数中带“|”符号会出错的问题，修正些其它小问题。

## 2014.04.28 v2.9.1

- 浏览器启动/退出可运行的外部程序，支持任意类型、任意多个。关闭外部程序时，将会同时关闭由外部程序启动的下一级子进程。
- 设置默认浏览器时不再显示对话框。
- 进一步减少 MyChrome 后台运行占用的系统资源（CPU 0%，内存 1MB 左右）。

## 2014.03.22 v2.9

- 增加 Chromium 分支 64 位版本支持（Chromium-Continuous-x64）。
- 所有路径支持 Windows 环境变量，特别地，%APP%代表 MyChrome 所在目录。
- 增加浏览器退出后运行外部程序等功能。
- 设置默认浏览器、锁定到任务栏后立即生效，不必再运行 MyChrome 一次。
- 增加“MyChrome 在后台运行直至浏览器关闭”选项。

## 2014.03.05 v2.8.3

- 解决 Windows Vista/7/8 等系统中，浏览器不能锁定到任务栏或者锁定后任务栏有两个浏览器图标的问题。
- 锁定方法：双击 MyChrome 启动浏览器 ==> 右键单击任务栏的浏览器图标，选择“将此程序锁定到任务栏” ==> 重新运行一次 MyChrome。

## 2014.02.24 v2.8.2

- 修复 Chrome 偏好设置文件损坏的问题。

## 2014.01.18 v2.8

- 解决更新过程中提取浏览器程序文件失败的问题。Chrome 可能改变了安装包的压缩格式，用精简版的 7za 解压会出错，MyChrome 2.8 版改用完整的命令行版 7z 解压。
- 增加启动浏览器时启动外部程序的功能。

## 2014.01.15 v2.7.4

- 修正 Windows 8 环境下“没有注册类”错误。
